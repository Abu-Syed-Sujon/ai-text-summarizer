#  AI Text Summarizer API

A modern and lightweight **AI-powered text summarization API** built with **FastAPI** and **Machine Learning**.  
It can summarize large text articles, research papers, or any content into concise summaries using advanced NLP models such as **Hugging Face Transformers** or **OpenAI GPT models**.

---

##  Features

- Summarizes long text content into short and meaningful summaries.
- Built with **FastAPI** for high performance.
- Supports both **local ML models** (Hugging Face) and **cloud-based APIs** (OpenAI).
- Simple RESTful API design for integration with front-end tools or apps.
- Easy to deploy on **Render**, **Railway**, or **Vercel**.

---

##  Tech Stack

- **Python 3.10+**
- **FastAPI**
- **Transformers / OpenAI API**
- **Uvicorn**
- **Pydantic**
- **Requests**
